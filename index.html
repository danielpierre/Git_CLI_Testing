Initialize Git Repository

  1. Create a new repository at github.com
  2. Create a project folder and type git init to initialize it as a git repository
  3. Type git remote add origin [URL]

Basic Git Workflow

  1. git status check status
  2. git add -A add files to staging area
  3. git status check status
  4. git commit -m commit files with a message
  5. git status
  6. git push origin master push commit to remote repository
